# space-invaders

Intel 8080 emulator written from scratch in ZIG, to run Space Invaders.
SDL is used as video interface.

![screenshot](https://github.com/oliverselinger/space-invaders/blob/main/screenshot.png?raw=true)

## Dependencies

* ZIG 0.8.1
* SDL2

## Build and Run

`zig build run`
